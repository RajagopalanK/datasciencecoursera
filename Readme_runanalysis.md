This markdown files provides an overview of run_analysis.R script

1. Download data from https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip

2. Unzip the data

3. set the working directory to setwd(/data/UCI/Dataset")

4. Execute the run_analysis.R R script in Rstudio

5. Read the tidy data set created using read.table("tidy_data.txt")
